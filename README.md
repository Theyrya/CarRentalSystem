Car Rental Management System

C#, .NET, MySQL

Developed a backend-driven car rental management application that allows rental business owners to manage vehicles, track rentals, and process returns through a centralized system.

Built the application using C# and .NET, focusing on backend logic and business workflows

Designed and implemented a MySQL relational database to manage cars, rental records, availability, and return history

Enabled owners to add, update, and remove vehicles from the rental inventory

Implemented rental and return workflows with automatic availability updates

Applied data validation and relational constraints to prevent invalid rentals and ensure data integrity

Focused on clean architecture, maintainable code, and real-world rental operations

This project simulates a real car rental business system and demonstrates backend development, database design, and business logic implementation.
